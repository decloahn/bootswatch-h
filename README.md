Bootstrap theme for Hoiio
==========


Usage
-----
Download all the necessary files from folder:

"bootstrap-hoiio"


"bootstrap-hoiio" folder structure:
+ css {subfolder}
	+ Contains 4 css files

+ img {subfolder}
	+ Contains images for icon-sprites

+js {subfolder}
	+ Contains all javascript files
